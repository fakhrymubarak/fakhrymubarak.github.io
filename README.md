## Getting Started

### Prerequisites

* Node.js (v18 or above)
* npm or Yarn or pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/fakhrymubarak/showcase-v2.git
cd showcase-v2

# Install dependencies
npm install
# or
yarn install
# or
pnpm install
```

### Development

```bash
# Start development server
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

### Build

```bash
# Build for production
npm run build
# or
yarn build
# or
pnpm build
```

### Deployment

```bash
# Deploy to GitHub Pages
npm run deploy
# or
yarn deploy
# or
pnpm deploy
```
